# 貪食蛇遊戲規格書

## 1. 專案概述
本專案是一個簡單的貪食蛇遊戲，使用 HTML、CSS 和 JavaScript 實現。遊戲在網頁瀏覽器中運行，玩家控制蛇吃食物以增加分數，遊戲結束條件為蛇撞到自己。

## 2. 功能需求
### 2.1 核心遊戲功能
- **蛇移動**: 蛇在 30x30 的網格中移動，支持上下左右方向。
- **邊緣傳送**: 蛇從一邊邊緣離開會從另一邊出現。
- **食物生成**: 隨機在網格中生成食物。
- **吃食物**: 蛇頭碰到食物時，分數增加，蛇身長度增加，速度加快。
- **碰撞檢測**: 蛇頭碰到蛇身時遊戲結束。
- **分數系統**: 顯示當前分數。
- **速度調整**: 吃食物後速度逐漸增加（最小間隔 10ms）。

### 2.2 控制功能
- **開始遊戲**: 點擊 "Start" 按鈕開始新遊戲。
- **暫停/繼續**: 點擊 "Pause" 按鈕暫停或繼續遊戲。
- **鍵盤控制**: 使用方向鍵控制蛇移動方向（不能反向）。

### 2.3 遊戲狀態
- **初始狀態**: 蛇長度 3，方向向右，速度 200ms/幀。
- **運行狀態**: 遊戲迴圈持續更新和繪製。
- **結束狀態**: 碰撞時顯示 "Game Over" 提示，停止遊戲。

## 3. 非功能需求
### 3.1 性能
- 遊戲在現代瀏覽器中流暢運行。
- 網格大小固定為 30x30，總共 900 個格子。

### 3.2 使用者體驗
- 簡單直觀的介面。
- 響應式設計（支援不同螢幕大小）。
- 黑色主題，白色蛇，紅色食物。

### 3.3 相容性
- 支援現代網頁瀏覽器（Chrome、Firefox、Safari 等）。
- 不依賴外部庫，純原生 JavaScript。

## 4. 技術規格
### 4.1 技術棧
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **架構**: 單頁應用，無後端。

### 4.2 檔案結構
- `index.html`: 主要 HTML 結構。
- `main.js`: 遊戲邏輯和事件處理。
- `style.css`: 樣式定義。

### 4.3 關鍵變數和常數
- `GRID_SIZE`: 網格大小 (30)
- `snake`: 蛇身座標陣列
- `direction`: 當前移動方向
- `food`: 食物座標
- `score`: 當前分數
- `speed`: 遊戲速度 (ms/幀)

### 4.4 主要函數
- `initBoard()`: 初始化遊戲板
- `gameLoop()`: 主遊戲迴圈
- `update()`: 更新遊戲狀態
- `draw()`: 繪製遊戲元素
- `startGame()`: 開始新遊戲
- `togglePause()`: 暫停/繼續

## 5. 使用者介面
### 5.1 佈局
- **中央**: 600x600 像素的遊戲板（30x30 網格）。
- **下方**: 控制面板，包含開始按鈕、暫停按鈕和分數顯示。

### 5.2 視覺元素
- **遊戲板背景**: 深灰色 (#161616)
- **蛇**: 白色 (#fff)
- **食物**: 紅色 (red)
- **按鈕**: 白色邊框，滑鼠懸停時反色

## 6. 遊戲規則
- 蛇初始長度為 3，向右移動。
- 使用方向鍵改變移動方向，不能直接反向。
- 吃到食物後分數 +1，蛇身 +1，速度加快。
- 蛇從邊緣離開會從另一邊出現。
- 蛇頭碰到蛇身時遊戲結束。
- 點擊 "Start" 重新開始遊戲。

## 7. 測試案例
- **正常遊戲流程**: 移動、吃食物、增加分數。
- **碰撞測試**: 蛇頭碰到身體時結束。
- **邊緣測試**: 蛇從邊緣傳送。
- **控制測試**: 方向鍵和按鈕功能正常。
- **暫停測試**: 暫停後繼續遊戲狀態正確。
